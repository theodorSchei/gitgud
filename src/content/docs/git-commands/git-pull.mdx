Dersom det er noen andre som har gjort endringer i prosjektet, og du ønsker å hente ned disse endringene til din lokale maskin, kan du bruke `git pull`. Dette henter ned endringer fra remote repostiory til lokalt repository.

## Eksempel

Du samarbeider med en medstudent på en oblig. Du har gjort ferdig din del av prosjektet, og din samarbeidspartner er ferdig med sin del. Dere ønsker å sette sammen prosjektet, og du bruker `git pull` for å hente ned endringene som din samarbeidspartner har pushet til GitHub.
